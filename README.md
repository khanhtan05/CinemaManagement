## ✨ Giới thiệu

**CineTicket** là ứng dụng đặt vé xem phim toàn diện, giúp bạn dễ dàng khám phá phim mới, chọn suất chiếu, chọn ghế ngồi đẹp và thanh toán chỉ trong vài giây.

Giao diện hiện đại, mượt mà, hỗ trợ **dark mode**, **real-time seat availability** và tích hợp nhiều rạp chiếu nổi tiếng.

---

## 🎥 Tính năng nổi bật

### Người dùng

- 📱 Duyệt phim đang chiếu & sắp chiếu (Poster đẹp + Trailer)
- 🔍 Tìm kiếm phim theo tên, thể loại, diễn viên
- 📅 Chọn ngày & suất chiếu linh hoạt
- 🪑 Chọn ghế ngồi theo thời gian thực (Ghế đã đặt sẽ hiện đỏ)
- 💳 Thanh toán nhanh (Momo, ZaloPay, VNPay, thẻ ngân hàng)
- 🎟️ Vé điện tử QR Code (Dễ dàng quét tại rạp)
- ⭐ Đánh giá & bình luận phim
- ❤️ Danh sách phim yêu thích
- 🛎️ Thông báo đẩy về suất chiếu sắp tới

### Quản trị viên / Rạp chiếu (Admin Panel)

- Quản lý phim, lịch chiếu, giá vé
- Quản lý ghế ngồi theo từng phòng
- Xem doanh thu & báo cáo
- Quản lý người dùng

## 🛠️ Công nghệ sử dụng

- **Frontend**: Flutter (hoặc React Native / React.js)
- **State Management**: BLoC / Riverpod / Redux
- **Backend**: Firebase / Node.js + Express + MongoDB
- **Database**: Firestore / PostgreSQL
- **Payment**: VNPay, Momo, ZaloPay SDK
- **Authentication**: Firebase Auth / JWT
- **Real-time**: Firebase Realtime Database / Socket.io
- **Other**: Dio, Shared Preferences, QR Code, Local Notifications

## 🚀 Cách cài đặt & chạy

### Yêu cầu

- Flutter 3.19+ (hoặc Node.js 20+ nếu là web)
- Android Studio / Xcode
- Firebase project

### Các bước

```bash
# Clone repository
git clone https://github.com/yourusername/cineticket.git
cd cineticket

# Cài đặt dependencies
flutter pub get

# Chạy ứng dụng
flutter run
```
